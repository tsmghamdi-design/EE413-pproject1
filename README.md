# Spoken Digit Classification Using Digital Signal Processing

## Project Overview

This project was developed for **EE 413: Applied Digital Signal Processing**.

The goal of this project is to classify spoken digits from audio recordings using Digital Signal Processing (DSP) techniques. The project uses the **Free Spoken Digit Dataset (FSDD)**, which contains short audio recordings of spoken digits from 0 to 9.

The project compares different feature extraction methods from the time domain and frequency domain, then uses machine learning classifiers to evaluate which approach gives the best classification performance.

---

## Dataset

The dataset used in this project is the **Free Spoken Digit Dataset (FSDD)**.

Dataset characteristics:

- Spoken digits from **0 to 9**
- WAV audio files
- Mono audio recordings
- Sampling rate of approximately **8 kHz**
- Short recordings, usually around 1 second
- Multiple speakers and multiple recordings per digit
- Clean audio with little or no background noise
